# 最大平均最小問題

名前（name）と得点（score）のオブジェクトが入ったユーザ(users)の配列を使って
下記の三つをコンソールに出力してください。

- 得点の平均点
- 最小得点の名前
- 最大得点の名前

## 入力例

```
const users = [
    {"name": "タロウ", "score": 60},
    {"name": "ジロウ", "score": 25},
    {"name": "サブロー", "score": 88},
    {"name": "シロウ", "score": 38},
    {"name": "ゴロー", "score": 49}
];
```

## 出力例

```
得点の平均点:**
最小得点の名前:**
最大得点の名前:**
```
