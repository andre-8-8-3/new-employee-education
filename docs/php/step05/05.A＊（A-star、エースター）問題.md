# A＊（A-star、エースター）問題


定義されたマップがあり、A＊アルゴリズムを使い
出発地点から目的地までの最短経路を見つけるプログラムを作成して
スタートからゴールまでの距離（移動数）とルートを出力してください。

マップの仕様は下記になります

  * 0 :移動可能
  * 1 :障害物
  * S :スタート
  * G :ゴール


## 入力例
```



$maps = [
    ["0", "0", "0", "0", "X", "0"],
    ["0", "X", "X", "0", "X", "0"],
    ["0", "X", "S", "0", "X", "0"],
    ["0", "G", "X", "0", "0", "0"],
    ["X", "0", "0", "X", "0", "0"],
    ["X", "X", "0", "0", "0", "X"]
];
```

## 出力例
```
距離:10
ルート:[
　　[ x => 2, y => 2],
　　[ x => 3, y => 2],
　　[ x => 3, y => 1],
　　[ x => 3, y => 0],
　　[ x => 2, y => 0],
　　[ x => 1, y => 0],
　　[ x => 0, y => 0],
　　[ x => 0, y => 1],
　　[ x => 0, y => 2],
　　[ x => 0, y => 3],
　　[ x => 1, y => 3]
]
```
