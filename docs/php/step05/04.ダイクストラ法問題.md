
# ダイクストラ法問題

隣接リストで表現されたグラフにおいて、
与えられた始点ノードから終点ノードまでの
最短距離と最短経路をダイクストラ法を用いて求める
プログラムを作成してください。
なお、グラフの辺には距離が与えられます。


## 入力例
以下のような重み付きグラフが与えられます。
```
$graph = [  
    1 => [2 => 7, 3 => 9, 6 => 14],
    2 => [1 => 7, 3 => 10, 4 => 15],
    3 => [1 => 9, 2 => 10, 4 => 11, 6 => 2],
    4 => [2 => 15, 3 => 11, 5 => 6],
    5 => [4 => 6, 6 => 9],
    6 => [1 => 14, 3 => 2, 5 => 9]
];
$start = 1;
$goal = 5;
```

## 出力例
```
最短距離:20
最短経路: 1 -> 3 -> 6 -> 5
```
